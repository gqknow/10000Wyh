澳大利亚：手提箱儿童遗骸追出母女双命案 被告认罪

2018-08-01 08:59环球网>>

　　【环球网报道 记者 赵怡蓁】瑞士媒体“瑞士资讯”7月31日援引法新社报道称，继装有小孩遗骸的手提箱在2015年被路人发现后，被控杀害一名年轻母亲与其幼女的一名澳大利亚男子于当地时间7月31日认罪，最终解开这起困惑警方多年的命案。

　　31日，43岁的荷登(Daniel James Holdom)在新南威尔士省最高法院出庭，承认于2008年杀害了当时20岁的皮尔斯-史蒂文生(Karlie Jade Pearce-Stevenson)和她的两岁女儿。

　　据报道，皮尔斯-史蒂文生的骸骨于2010年在新南威尔士省贝朗格罗省立森林(Belanglo State Forest)被人发现。这个恶名昭彰的地点在1990年代发生连续杀人事件，挖出7名背包客尸体。

　　警方一直无法查出被害人身分，直到2015年，皮尔斯-史蒂文生小孩的遗骸在约1100公里外南澳大利亚省一个小镇，被发现藏在公路附近的一只手提箱内。小孩遗骸被发现后，警方报案热线接获的两则情报最终协助调查人员通过DNA检验，查出了受害母女的身份。

　　澳大利亚《每日电讯报》引述法院文件称，荷登与皮尔斯-史蒂文生有过短暂的一段恋情，他被控踩踏她的喉咙，踩断她的气管，然后弃尸森林。警方还指控荷登保留皮尔斯-史蒂文生的多张尸体照片，充当“战利品”。

　　荷登表示，数几天后，他开车带皮尔斯-史蒂文生的女儿到南澳省其祖母住处，但他决定杀害她，将尸体装进手提箱，然后丢弃在公路旁。

　　报道称，法院预定9月底对此案进行宣判。
